\begin{frame}{为什么列书单?}
  \begin{enumerate}
    \setlength{\itemsep}{8pt}
    \item 我感受到了大家强烈的求知欲
    \item 希望大家在本科阶段多读一些书
    \item 有书读多快乐啊
  \end{enumerate}

  \vspace{0.80cm}
  \begin{quote}
    \centering
    ``列书单就是把自己也没读过的书\\[3pt]
    推荐给同样不会去读的其他人,\\[3pt]
    以达到大家都已读过这些书的假象。''
  \end{quote}
\end{frame}
