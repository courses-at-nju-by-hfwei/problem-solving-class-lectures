	前提如下:

	\begin{enumerate}[(1)]
	  \item 每个人或者喜欢美剧, 或者喜欢韩剧 (可以同时喜欢二者);
	  \item 任何人如果他喜欢抗日神剧, 他就不喜欢美剧;
	  \item 有的人不喜欢韩剧。
	\end{enumerate}
	结论: 有的人不喜欢抗日神剧。
    \end{enumerate}
  \end{exampleblock}
